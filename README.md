# Task Management System

## 📌 Overview
A simple JavaFX desktop application that allows users to manage and organize their tasks.  
The system supports adding new tasks, filtering tasks, counting tasks, and displaying summaries.

----- Features
- Add new tasks with title, status, date, and user name.
- Display the **total number of tasks**.
- Show the number of **open** and **closed** tasks.
- Identify the **top user** who added the highest number of tasks.
- Filter tasks by user name.
- Display the first 4 tasks sorted by date.
- Find special tasks (titles starting with "a" and length = 7).
- Count tasks for a specific user.
- Simple menu bar to change font style, size, and exit the program.
- 
 ---------- Project Structure
- **src/** → Source code (controllers, views, models).
- **data/** → Contains `task.csv` file with sample tasks.
- **View/** → FXML and CSS files for GUI design.
- **model/** → Task class definition.
- **Controller/** → Main controller logic (`MainfxmlController.java`).
